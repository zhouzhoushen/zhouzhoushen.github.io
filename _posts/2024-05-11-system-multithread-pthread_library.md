PThread is alternative to write multithread programs. You can use either C++'s `thread` or `pthread`. C++'s `thread` provides straightforward APIs, which is friend to programmers. `pthread` is a little bit more complex, while providing more fine-grained control. You can refer to this [article](https://blog.csdn.net/zhouqt/article/details/127526970) to get a quick start.
