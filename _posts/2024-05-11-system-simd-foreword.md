SIMD means *Single Instruction Multiple Data*. Using SIMD instruction set such as MMX, SSE or AVX, you can operate on multiple variables with single instruction, thus saving time.

You can read articles like [this](https://blog.csdn.net/qq_32916805/article/details/117637192) to learn what it is and articles like [this](https://zhuanlan.zhihu.com/p/457505686) for a quick start.
